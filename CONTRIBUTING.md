# Contributing to FunZone

Thanks for your interest in helping FunZone evolve!

## Ways to Contribute
- Add new FunZone Node firmware modules
- Create Node-RED flows
- Improve documentation or examples
- Submit wiring diagrams or hardware ideas
- Report bugs or propose improvements

## Process
1. Open an Issue describing your idea or bug.
2. Fork the repo and create a feature branch.
3. Submit a PR with a clear description.
4. We review and merge.

## Code Style
- Keep firmware readable and simple.
- Use consistent naming for MQTT topics.
- Include comments for hardware pinouts.
